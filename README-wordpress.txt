WordPress upload instructions:

1. Copy the file arise-registration-plugin.php into your WordPress site's wp-content/plugins/ folder.
2. Rename it to arise-registration-plugin.php if needed.
3. Go to WordPress Admin > Plugins and activate "Arise Registration".
4. Add this shortcode to any page or post:
   [arise_registration_form]
5. Replace YOUR_APPS_SCRIPT_WEBAPP_URL inside the plugin file with your Apps Script Web App URL to save submissions to Google Sheets.
